{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} WingVortex is a simple extension to add support for Project Wingman to the Vortex mod manager. With this extension you can install, remove and manage your Project Wingman mods using Vortex, perfect for players already familiar with Vortex. **This project is unofficial!** We have no association with Nexus Mods nor with Sector D2 and is only a community project. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-tools" title="Full Vortex integration" %}} Manage your Project Wingman mods the same way you manage any other Vortex-supported game. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/agc93/wingvortex" %}} WingVortex is fully open-source and open to community contributions and fixes! {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cloud-download-alt" title="Supports Most Mods" %}} You should be able to install almost any mod for Project Wingman. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
